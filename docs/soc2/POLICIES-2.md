# SOC 2 Policy Documents — Set 2

https://linear.app/wopr/issue/WOP-931/soc-2-policy-documents-change-management-incident-response-risk

## Policies

1. **Change Management Policy** — code review requirements, deployment approvals, rollback procedures, emergency change process
2. **Incident Response Policy** — severity classification (SEV1-3), escalation paths, communication templates, postmortem requirements
3. **Risk Assessment Policy** — risk identification process, scoring methodology (5x5 matrix), risk register (R-001 through R-010)
4. **Vendor Management Policy** — vendor security tiers (1/2/3), vendor inventory (14 vendors), DPA requirements, annual review process
